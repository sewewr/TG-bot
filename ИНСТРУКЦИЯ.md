# Инструкция по запуску бота

## Шаг 1 — Создай бота в Telegram

1. Открой Telegram, найди @BotFather
2. Напиши `/newbot`
3. Придумай имя (например: `Курьер СПб`) и юзернейм (например: `courier_spb_bot`)
4. Скопируй токен — выглядит так: `7123456789:AAFxxxxxxxxxxxxxxx`

---

## Шаг 2 — Узнай свой chat_id

1. Найди в Telegram бота @getmyid_bot
2. Напиши ему `/start`
3. Он пришлёт твой числовой ID, например: `123456789`

---

## Шаг 3 — Заполни config.py

Открой файл `config.py` и замени:
- `ВСТАВЬ_ТОКЕН_СЮДА` → токен от BotFather
- `ВСТАВЬ_СВОЙ_CHAT_ID` → твой числовой chat_id

Пример готового config.py:
```python
BOT_TOKEN = "7123456789:AAFxxxxxxxxxxxxxxx"
ADMIN_CHAT_ID = 123456789
```

---

## Шаг 4 — Замени ссылки в bot.py

Открой `bot.py` и найди строки с комментарием `# TODO`:

1. **Партнёрская ссылка** (функция `go_register`):
   ```
   https://eda.yandex.ru/couriers  ← замени на свою ссылку из партнёрского кабинета
   ```

2. **Твой юзернейм** (функция `write_to_human`):
   ```
   @ваш_юзернейм  ← замени на свой @username в Telegram
   ```

---

## Шаг 5 — Запусти бота

### Вариант А: на своём компьютере (для теста)

Установи Python 3.10+ если не установлен: https://python.org

Открой терминал в папке с файлами и выполни:
```bash
pip install -r requirements.txt
python bot.py
```

Бот будет работать, пока открыт терминал.

---

### Вариант Б: на Railway (бесплатно, работает 24/7) — РЕКОМЕНДУЕТСЯ

1. Зарегистрируйся на https://railway.app (можно через GitHub)
2. Создай новый проект → "Deploy from GitHub repo"
   - Или: "Empty project" → загрузи файлы вручную
3. В разделе **Variables** добавь:
   - `BOT_TOKEN` = твой токен
   - `ADMIN_CHAT_ID` = твой chat_id
4. В разделе **Settings → Start Command** укажи:
   ```
   python bot.py
   ```
5. Deploy — бот запустится автоматически

---

### Вариант В: на VPS (если есть сервер)

```bash
# Подключись к серверу по SSH, залей файлы
git clone или scp

# Установи зависимости
pip install -r requirements.txt

# Запусти в фоне через screen
screen -S courier_bot
python bot.py
# Ctrl+A, затем D — выходим из screen, бот продолжает работать
```

---

## Проверка работы

После запуска найди своего бота в Telegram по юзернейму и напиши `/start`.
Ты должен увидеть приветственное сообщение.

При заполнении всех шагов тебе придёт уведомление в личку с данными лида.

---

## Если что-то не работает

| Ошибка | Решение |
|--------|---------|
| `Unauthorized` | Неверный токен в config.py |
| `Chat not found` | Неверный ADMIN_CHAT_ID или ты не написал боту /start |
| `ModuleNotFoundError` | Не установлены зависимости: `pip install -r requirements.txt` |
| Бот не отвечает | Проверь, запущен ли скрипт (`python bot.py` в терминале) |
